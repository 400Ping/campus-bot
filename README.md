
# Campus Assistant LINE Bot + Website (Flask)

Commands in LINE:
- /help
- /schedule today|tomorrow|week
- /note <text>
- /review today
- /news add <kw> | /news list | /news remove <kw>
- /translate on|off

Website pages:
- /web/schedule (view)  |  /web/schedule/manage (add/import)
- /web/notes (list on home)  |  /web/notes/manage (add via form)
- /web/news (keywords add/remove)
- /web/review (generate today's review pack)
- /web/settings (toggle translate)
